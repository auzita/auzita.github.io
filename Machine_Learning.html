Pattern Based Security Using Machine Learning Techniques

I worked on a project titled "Pattern Based Security Using Machine Learning Techniques" during my final year of computer engineering. I worked as part of a team of four. Our project was an attempt to reduce false alarm rate for Intrusion Detection System by using Machine Learning algorithm. Our aim was to design an IDS by using machine learning which could meet the demands of Reducing False Alarm Rate (FAR) along with higher detection rate. Many types of IDS already exist in the world that provide assistance at different stages of project development. But one problem that is commonly observed is that of a high False Alarm Rate. Our software should be able to assist the developers in this department greatly along with the individual stage support.
We were able to achieve an accuracy rate of 98.78% and reduce the false alarm rate to 1.22%
